# Android Phỏng Vấn

Bộ câu hỏi ôn tập Android/Kotlin cho phỏng vấn.

## Cấu trúc

```
/{topic}/
  - {issue}.md
```

Mỗi file có frontmatter metadata (ID, Date, Tags) + nội dung chi tiết.

## Topics

- `livedata/` - LiveData
- `viewmodel/` - ViewModel
- `coroutines/` - Coroutines
- *(thêm mới khi cần)*

## Hướng dẫn

Clone repo và đọc theo topic:
```bash
git clone https://github.com/trienhieunu/Android-pv.git
```

Mỗi topic folder chứa các file issue theo thứ tự ID (001, 002, 003...).
