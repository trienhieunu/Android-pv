# Android Phỏng Vấn (Interview Questions)

Bộ câu hỏi ôn tập Android/Kotlin cho phỏng vấn.

## Cấu trúc

Mỗi chủ đề có một file riêng theo format: `{chủ đề}.md`

Mỗi file chứa:
- **Câu hỏi** - Câu hỏi phỏng vấn
- **Câu trả lời** - Giải đáp chi tiết
- **Tình huống ví dụ** - Code minh họa và timeline

## Chủ đề hiện có

- `livedata-config-change.md` - Vấn đề LiveData khi config change/rotate
- *(Thêm mới khi có câu hỏi khác)*

## Quy tắc cập nhật

Khi có câu hỏi mới:
1. **Kiểm tra xem có file cùng chủ đề chưa**
2. Nếu có: update file hiện có (thêm câu hỏi vào file đó)
3. Nếu chưa: tạo file mới theo format `{chủ đề}.md`

Ví dụ:
- Câu hỏi về `ViewModel` → kiểm tra file `viewmodel.md`
- Câu hỏi về `Coroutines` → kiểm tra file `coroutines.md`
- Nếu không có → tạo file mới

## Tags

Mỗi bài viết có tags để dễ tìm kiếm:
- VD: `LiveData`, `ViewModel`, `Config Change`, `Observer Pattern`, `Lifecycle`

## Hướng dẫn sử dụng

1. Clone repo: `git clone https://github.com/trienhieunu/Android-pv.git`
2. Đọc file theo chủ đề cần ôn tập
3. Thực hành với code trong "Tình huống ví dụ"

---

**Mục tiêu:** Tổng hợp kiến thức Android/Kotlin cho phỏng vấn một cách hệ thống.
